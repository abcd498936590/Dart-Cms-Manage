# 这是Dart-Cms的后台管理系统页面项目.
> 技术栈 vue + vue-router + element-ui

## Dart-Cms

[Dart-Cms完整项目地址](https://github.com/abcd498936590/Dart-Cms)


## 安装

``` bash
# 安装依赖
npm install

# 项目运行
npm run dev

# 打包
npm run build
```
